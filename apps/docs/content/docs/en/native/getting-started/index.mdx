blakeUI Native is a React Native component library built on [Tailwind v4](https://tailwindcss.com/blog/tailwindcss-v4) via [Uniwind](https://uniwind.dev/) and modern mobile development technologies. Every component comes with smooth animations, polished details, and built-in accessibility—ready to use, fully customizable.

## FAQ

**Is blakeUI Native free?**
Yes, completely free and open source under the <ApacheLicense />.

**Is it production-ready?**
Yes, blakeUI Native is production-ready and actively used in production apps.

**Can I customize the components?**
Yes! Update default styles, animations or compose component parts differently. Every slot is customizable.

**Does it work with TypeScript?**
Fully typed with excellent IDE support and autocompletion.

**What about accessibility?**
Accessibility follows mobile development best practices with proper focus management, touch accessibility, and screen reader support built into every component.

blakeUI Native is released under the <ApacheLicense />.
